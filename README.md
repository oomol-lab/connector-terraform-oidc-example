# connector-terraform-oidc-example

Terraform examples for configuring OOMOL OIDC federation on supported cloud
providers.

The content in this repository is intended for the three STS providers available
from <https://console.oomol.com/connections>:

- `aliyun/` for the Alibaba Cloud STS provider
- `aws/` for the AWS STS provider
- `gcp/` for the Google Cloud STS provider
